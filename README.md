## 📖 About

**Zero Bot** is a WhatsApp bot built using the Baileys library, designed specifically for group management. It includes features like tagging all members, muting/unmuting, games, and many more. It is designed to help admins efficiently manage large WhatsApp groups.

This repository is a continuously updated custom version of the [original Knightbot](https://github.com/mruniquehacker/Knightbot) project, featuring various upgrades and optimizations by Hunter Zen.

## 🛠️ Local Setup & Installation

If you prefer to run the bot locally on your machine or VPS via terminal, follow these steps.

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your system
- [Git](https://git-scm.com/) installed
- A WhatsApp account

### Step-by-Step Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/zero-one-community/zz.git
   cd zz
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Run the bot:**
   ```bash
   node index.js
   ```

4. **Connect your WhatsApp:**
   Once the bot starts, a QR code will appear in the terminal (or use the Pair Code method). Scan this QR code using the **Linked Devices** feature in your WhatsApp app to connect your account.

## ⚙️ Features

- **Tag All Members:** Mention everyone in a group with `.tagall`.
- **Admin Control:** Restrict sensitive commands to group admins only.
- **Interactive Games:** Play games like Tic-Tac-Toe directly in the chat.
- **Text-to-Speech:** Convert text to voice messages using `.tts`.
- **Sticker Creation:** Easily create stickers from images using `.sticker`.
- **Group Safety:** Automatic anti-link detection to prevent spam.
- **Warning System:** Warn and manage group members efficiently.
